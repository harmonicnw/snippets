# Harmonic jQuery Mobile Refresh Widgets 
### Version: 1.0.0
	
## Dependencies

* jQuery (optimized for 10.2)	
	
## Usage

```
$("body").hmcJqmRefresh(['header','listview']);

```

### Options	

* **[array]** : *( optional | [array] ) options: "button", "header", "listview", "select"
