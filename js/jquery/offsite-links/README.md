# Harmonic Offsite Links jQuery plugin
### Version: 1.0.3

## Dependencies

* jQuery (optimized for 10.2)

## Usage

```
$('body').hmcOffsiteLinks({
		site : "harmonicnw.com"
	});
```

### Options

* **site** : *( optional | string | default = document.location.host )* string that identifies this site
