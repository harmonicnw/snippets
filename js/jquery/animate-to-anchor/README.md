# Harmonic Animate to Anchor jQuery pluggin. 
### Version: 1.0.0
	
## Dependencies

* jQuery (optimized for 10.2)	
	
## Usage

```
$("nav").hmcAnimateToAnchor( {bufferTop : 40} );

```

### Options	

* **bufferTop** : *( optional | integer | default = 0 ) identifies the top buffer for pages that have a sticky header.
